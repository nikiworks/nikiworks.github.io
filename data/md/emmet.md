逆引き   | 記号 | 記法 | 出力結果
---------|------|------|--------------------

         |      |      |
         |      |      |
         |      |      |
         |      |      |
         |      |      |

  記号	役割	記述例
  #	ID名の指定	div#idName
  .	class名の指定	div.className
  >	要素を入れ子にする	div>p
  +	要素を同階層に展開する	div+p
  ^	一階層上に展開	div>p^ul
  *	要素を複数展開	li*3
  {}	テキストの挿入	p{テキスト}
  $	連番をつける	li.item-$*3
  !	HTMLの雛形	!
